    Funciones

    Una función matemática es una relación que asigna a cada elemento de un conjunto un único elemento de otro conjunto. Las funciones son fundamentales en matemáticas y se utilizan para describir diversas relaciones y fenómenos.

    Ejemplos de Funciones:

    • Función Cuadrática: f(x) = x² - 4x + 3. La gráfica de esta función es una parábola.
    • Función Lineal: f(x) = 2x + 1. Esta función representa una línea recta.
    • Función Exponencial: f(x) = 2^x. Esta función crece rápidamente a medida que x aumenta.
    • Función Seno: f(x) = sin(x). Esta función se utiliza para describir ondas periódicas.

    Propiedades de las Funciones:

    • Dominio: Conjunto de valores de entrada (x) permitidos.
    • Rango: Conjunto de valores de salida (f(x)) resultantes.
    • Continuidad: Una función es continua si no presenta saltos o interrupciones en su gráfica.
